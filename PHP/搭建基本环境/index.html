一.PHP开发环境安装

这里使用Appserv 9.3.0，PHP+MySQL+Apache，推荐Win10运行。

原来软件下载链接在自己跳墙后面的SourceForge里，网盘链接：https://wws.lanzous.com/iuTy8hcivgj，下载后安装。Appserv支持

  • Apache 2.4.41

  • PHP 7.3.10

  • MySQL 8.0.17

  • phpMyAdmin 4.9.1

  • Support TLS,SSL or https

(仅支持64位)
